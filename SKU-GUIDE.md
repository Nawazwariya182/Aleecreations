# Alee Creation's - SKU Generation Guide

## SKU Format

```
[CategoryCode][MaterialCode][ThemeCode][ID]-[VariantCode]
```

**Example:** `CHCRLX001-GD` = Charms + Crystal + Luxury + ID 001 + Gold

---

## Code Reference Tables

### Category Codes (2 letters)

| Code | Category           |
|------|--------------------|
| CH   | Charms             |
| LS   | Long Strip Charms  |
| ST   | Sets               |
| BP   | Bag Pins           |

---

### Material Codes (2 letters)

| Code | Material / Element        |
|------|---------------------------|
| CR   | Crystal (Faceted Crystal Beads) |
| PL   | Pearl (Faux Pearls)       |
| AC   | Acrylic (Acrylic Beads)   |
| RS   | Resin                     |
| GL   | Glass                     |

---

### Theme Codes (2-3 letters)

| Code | Theme / Style               |
|------|-----------------------------|
| LX   | Luxury (Elegant, Premium)   |
| CT   | Cute (Kawaii, Playful)      |
| MN   | Minimal (Simple, Clean)     |
| PS   | Pastel (Soft Colors)        |
| FL   | Floral (Flowers, Nature)    |

---

### Variant / Color Codes (2 letters)

| Code | Color / Version   |
|------|-------------------|
| GD   | Gold / Amber Gold |
| SL   | Silver            |
| BK   | Black             |
| WH   | White / Clear     |
| PK   | Pink              |
| BL   | Blue              |
| PL   | Purple / Lavender |
| ML   | Multi / Iridescent|
| PS   | Pastel            |
| BZ   | Bronze            |

## How to Generate a New SKU

1. **Identify the Category** → Use the 2-letter category code
2. **Identify the Primary Material** → Use the 2-letter material code
3. **Identify the Theme/Style** → Use the 2-3 letter theme code
4. **Assign the Product ID** → Pad with leading zeros to 3 digits
5. **Identify the Dominant Color** → Use the 2-letter variant code

### Example: New Product

**Product:** "Rose Quartz Butterfly Charm" (ID: 58)
- Category: Charms → `CH`
- Material: Crystal → `CR`
- Theme: Luxury → `LX`
- ID: 58 → `058`
- Color: Pink → `PK`

**Result:** `CHCRLX058-PK`

---

## Notes

- **ML (Multi)** is used for iridescent, holographic, or rainbow products
- **PS (Pastel)** can be used for both theme and variant when the product is predominantly pastel-colored
- **LX (Luxury)** is the default theme for elegant, premium, or sophisticated designs
- **CT (Cute)** is used for playful, kawaii, or whimsical designs
- **MN (Minimal)** is for simple, clean designs without elaborate ornamentation
